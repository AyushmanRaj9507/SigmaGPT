# IntelliChat 🚀
A MERN based ChatGPT replica implemented from scratch using OpenAI.

A modern AI chatbot built using React, Node.js, Express and Groq API.

## Features

- Multi-thread conversations
- Dark / Light theme
- Search chats
- Delete chats
- Export chats to PDF
- Download chats as TXT
- Responsive UI
- Chat history persistence
- Groq AI integration

## Tech Stack

Frontend:
- React
- Context API
- CSS

Backend:
- Node.js
- Express

Database:
- MongoDB

AI:
- Groq API

## Installation

### Frontend

npm install
npm run dev

### Backend

npm install
npm run start
