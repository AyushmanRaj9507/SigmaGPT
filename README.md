# IntelliChat 🚀

AI-powered chatbot built using React, Node.js, MongoDB and Groq AI.

## Features

- AI Chat with Groq
- Chat History
- Multiple Threads
- PDF Export
- Dark / Light Theme
- Responsive Design
- Search Chats
- Delete Chats

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express.js
- MongoDB

AI:
- Groq API
